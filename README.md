# Mycloud (Personal Cloud Storage)

icloud.py => high level view of cloud storage using FastAPI:::  main.rust => main project, Hash and Security will be applied later

Using Rasberry Pi and SSD to store anything.
## purpose
I hit the limit of the iCloud because of Goodnotes, and my phone suggest me to upgrade my plan, so I decided to make my own one.
Figuring out how to connect them to my storage (might not be possible, but still work for photo storage)

This topic was good material learning, I wanted to understand how web works, TCP, HTTP, Rest etc...
